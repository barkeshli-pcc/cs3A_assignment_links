Generate 1000* integers between 1* and 6* inclusive. Place them in a text file.

Read these numbers into an array of ints.

write a function to find the frequency of the occurrence of each number and store it in a two dimensional array of ints.

Generate a report that will tell the user the frequency of each number along with the percentage to the total numbers

 

*these numbers should be stored in constants to make it easy to change them in the program.

Don't forget your top/down design principles.
