Extra Credit: Linked List Functions

Implement a simple singly linked list and demonstrate common operations.

Requirements

  • Implement insert at head.
  • Implement insert after a given node.
  • Implement search.
  • Implement print.
  • Free all dynamically allocated memory before exiting.
  • Demonstrate each operation with labeled output.

General Expectations

  • Organize your solution into multiple functions.
  • Compile without warnings or errors.
  • Produce clear, labeled output.
  • Include comments explaining what each section demonstrates.
  • The goal is to explore the topic and demonstrate understanding, not to build a large application.
