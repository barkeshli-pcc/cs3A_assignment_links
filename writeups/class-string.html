class String:

Wrap your pointer based cstring classes in a class called String. 

We will try to mimic the behavior of the std::string class. We will need all the common functions of string:

You already have all (or most of these functions) written. Include the header file that contains the pointer version of your cstring and call the functions from the member function of your String class.

Don't forget your Big 3!

Do not call your String header file string.h!!
