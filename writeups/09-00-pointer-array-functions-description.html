Extra Credit: Pointer Array Functions

Implement reusable functions that operate on arrays using pointers rather than indexing whenever practical.

Requirements

  • Implement several functions such as print, search, append, copy, reverse, count, min/max, or compare.
  • Call each function from main().
  • Display clear output before and after each operation.
  • Use meaningful test data and comments.

General Expectations

  • Organize your solution into multiple functions.
  • Compile without warnings or errors.
  • Produce clear, labeled output.
  • Include comments explaining what each section demonstrates.
  • The goal is to explore the topic and demonstrate understanding, not to build a large application.
