Extra Credit: C++ Inheritance Playground

Write a C++ program exploring inheritance, virtual functions, and how base and derived classes behave when used through pointers and references.

Requirements

  • Create one base class and at least two derived classes.
  • Include constructors and destructors in each class (print messages to observe execution order).
  • Use private, protected, and public members to explore access rules.
  • Override at least one virtual function in the derived classes (do not use pure virtual functions).
  • Create derived objects and store them using base-class pointers.
  • Demonstrate runtime polymorphism by calling virtual functions through base-class pointers.
  • Include at least one non-virtual function and show how its behavior differs from virtual functions.
  • Write at least one function that takes a base-class pointer or reference.
  • Use static_cast to cast a base-class pointer back to a derived class pointer and call a derived-specific function.
  • Clearly label all output so each concept is easy to follow.
  • Comment each section of code explaining what is being demonstrated.

General Expectations

  • Organize your solution into multiple classes and functions.
  • Compile without warnings or errors.
  • Produce clear, labeled output showing program behavior.
  • The goal is to explore how inheritance works in C++, not to build a large application.
