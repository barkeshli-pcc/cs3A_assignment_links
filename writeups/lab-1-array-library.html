Lab 1: Array Library

 

 

The following functions manipulate an int array (not a cstring) Implement and fully test each function. All functions will have to be passed, at the minimum, the array and the size of the array as arguments.

 

template <typename T>
void _array_print(T arr[], int size);

template <typename T>
void _array_copy(T dest[], T src[], int how_manuy);

template <typename T>
void _push_back(T arr[], int &size, const T &push_me);

template <typename T>
void _pop_back(T arr[], int &size, T &pop_me);

template <typename T>
int _find(T arr[], int size, const T &find_me);

 

older versions:

 

The following functions manipulate an int array (not a cstring) Implement and fully test each function. All functions will have to be passed, at the minimum, the array and the size of the array as arguments.

array_print: prints the array. iterate through the array and print each element.

array_copy: copy one array to another.

push_back: add one element to the end of the array

pop_back: remove one element from the end of the array

find: given an item, return the position of the item or return -1 if not found.
