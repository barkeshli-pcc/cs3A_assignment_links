Design a class to be used to abstract a two dimensional pointer array:

This class will be used to implement the Labs and Plane assignment.

For the easy version, a static array of int pointers, define a const NUM_OF_ROWS=5 and have a member variable table:

int* table[NUM_OF_ROWS];

the constructor will take an array of ints that represents the sizes of each row (number of ints in each row)

 

You will need most (if not all) of these functions

allocate () //allocate a one dimensional int array

allocate() //allocate a two dimensional int array

read //read contents of row and col

write //write item in row and column

CellExists //verify that a row and column exists

search //search for a key. return the pointer to the found cell, or null if not found. if found, row and col will contain the location of the found cell

print //print all the elements of the two dimensional array.

overload the bracket operator...

 

EXAMPLE:

...

int A[] = {3,4,5,6,7};

TowDint T2(A);

T2.Write(1,1,75); //T2[1][1] = 75;
