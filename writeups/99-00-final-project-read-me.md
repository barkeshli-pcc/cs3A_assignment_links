## Features Implemented

- [ ] Equation history in the sidebar, newest entry on top
- [ ] Clicking a history entry displays its graph
- [ ] History is read from disk on startup; `sin(x)` is the default entry
      when no history file exists yet
- [ ] Equation entry is space-insensitive and case-insensitive
- [ ] Trig functions: `sin`, `cos`, `tan`, `asin`, `acos`, `atan`
- [ ] `sqrt`, `abs`, `Pi()`
- [ ] Zoom in / out (`Enter` / `Shift+Enter`) via domain & range — no SFML zoom
- [ ] Pan (arrow keys[, mouse wheel / trackpad pinch]) via domain & range — no SFML view transforms
- [ ] Square work area with equal x/y pixel scale
