Linked List Functions: 

I suggest you study the algorithms you are going to use on paper very carefully before you start to write code. (Never write pointer or linked list related code without first drawing boxes and arrows.) Linked list bugs (pointer related bugs in general) tend to be illusive and hard to find and debug. Best strategy is avoidance.

Selection Sort, Merge (of two sorted lists), UniqueMerge, SplitList, AddLists, Subtract Lists. Although we will write these eventually so you can go ahead and do them now if you'd like. Makes your life easier later.

 

  • PrintList:                Prints the items in list. Uses setw( ) to space the items on the output stream
  • SearchList             Takes a value (key) and returns the pointer to the node containing that item. Null if not found
  • InsertHead            Takes a number (insertThis) and inserts it at the head of the list
  • PreviousNode:      Takes a pointer to an item in the list (previousToThis) and returns a pointer to the node before the designated node. Returns NULL if the node is the first node in the list.
  • InsertBefore:         Takes aa value (insertThis), and a pointer to a node in the list (beforeThis) and inserts the value before the designated node.
  • InsertAfter:            Takes a value (insertThis) and a pointer to a node in the list (afterThis) and inserts the value after the designated node
  • Delete:                  Takes a pointer to a node in the list (deleteThis) and removes that node from the list. Deletes the node and returns the item contained within.
  • WhereThisGoes:   Takes a value (item) and returns a pointer to the node after which the item belongs in the list.  Assumes the list is sorted. Returns NULL if the item belongs at the head of the list.
  • CopyList:               Takes a list (head) and returns a pointer to a linked list that is a copy of this list.
  • ClearList:               Takes a list (head) and deletes all the nodes in the list
  • At:                          Takes a position (n) and returns a pointer to the nth position in the list
