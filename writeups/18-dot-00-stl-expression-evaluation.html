STL: Expression Evaluation

Note: this is a tentative description, we will refine it as we go forward with this project.

One of the classic use cases of Stacks and Queues is expression evaluation.

 

Use the C++ Standard Template Library data structures (stack, queue..) to implement an Eval Class to evaluate an expression entered as postfix or as infix:

 

Oh, and... Don't forget to build the Expression Tree!


 

infix: (-1*b + ((b^2-4*a*c)^0.5))/2*a

postfix: -1 b * b 2 ^ 4 a * c * - 0.5 ^ 2 a * /

 

your stacks and queues (and yes, the tree) will have the ability to push and pop variety of types: ints, doubles and even functions and variables.

 

 NOTE: You will use true object oriented methods (Inheritance, Virtuality, Metamorphism) to solve this problem.
