Extra Credit: Two-Dimensional Pointers

Write a C++ program exploring pointers to pointers and/or dynamically allocated 2D arrays.

Requirements

  • Create and initialize a 2D structure.
  • Access elements using pointer notation.
  • Pass the structure to functions.
  • Modify data through pointers.
  • Print the contents in table form.
  • Comment each demonstration.

General Expectations

  • Organize your solution into multiple functions.
  • Compile without warnings or errors.
  • Produce clear, labeled output.
  • Include comments explaining what each section demonstrates.
  • The goal is to explore the topic and demonstrate understanding, not to build a large application.
