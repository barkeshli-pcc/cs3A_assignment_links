9.1 - Add Big Int

For this project, rewrite (or write for the first time,) the add big int project from chapter 7:

  • You will use dynamic arrays so that any size number can be added.
  • Use ONLY the pointer notation:
    • you will NOT use the [ ] operator. 
    • you will not use the ugly *(p + i) notation.
    • use if you need to access a cell in the array, get a walker and point to the cell you want to access
    • you may use walker = walker + 1 or walker = walker + i
  • The objective here is to get you accustomed to pointers to the point where you are not simply translating array notation to pointer notation. I want you to think in pointers.
  • Test your program with a simple pair of numbers.
  • Test your program with overflow
  • In your output, indicate that you have checked the correctness of  your calculation.
    • Do not simply add 1394879595 and 92929484020249 and give me a number that is hard to verify
